## P0 found and fixed (Janus)

**Bug:** `collectInlineChildren` / list item / table cell called  
`convertInline(child, undefined, { inherited })` and **dropped** `options.imageBuffers`.

**Effect:**  
- base64 inline images worked (no cache needed)  
- **remote `<img>` inside `<p>`, `<li>`, `<td>` always skipped** after successful prefetch  
- Confirmed: `<p>Before <img src=https://…> after</p>` → media `[]`, no `w:drawing`

**Fix:** Thread `{ ...options, inherited }` from:
- `src/blocks.js` `collectInlineChildren` + bare text path  
- `src/lists.js` list item inlines  
- `src/tables.js` cell paragraphs  

**Regression:** `testInlineRemoteImageUsesPrefetchBuffers` in `test/images.test.js`.
